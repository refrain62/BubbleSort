# Bubble Sort  

## 平均計算量： <img src="https://latex.codecogs.com/gif.latex?\O(n^2)" />  

* 先頭から順に要素の終わりまで順に次の要素と比較し、次の要素が小さい場合は要素を入れ替えていく
* 先頭の要素を1つずつずらしながら上記の処理を要素の最後まで繰り返す
